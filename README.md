# Run it locally

1. Clone the repo

   ```sh
   git clone https://github.com/retconned/treeverse-web
   ```

2. Change directory to the project folder

   ```sh
   cd treeverse-web
   ```

3. Run the app locally

   ```sh
   yarn dev
   ```
